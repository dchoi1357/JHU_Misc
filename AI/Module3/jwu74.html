CSP: Map Coloring¶

In this programming assignment, you will be using your new understanding of Constraint Satisfaction Problems to color maps. As we know from the Four Color Theorem any division of a plane into contiguous regions can be colored such that no two adjacent regions are the same color by using only four colors.

From the book, we know that we can translate this problem into a CSP where the map is represented as a planar graph and the goal is to color all the nodes such that no adjacent nodes are colored the same color.

As with most AI problems, this requires us to figure out how best to represent the problem--and the solution--given the high and low level data structures and types at our disposal. For this problem, we'll settle on a Dict which contains at least two keys: "nodes" which is an ordered List of Strings that represents each node or vertex in the planar graph and "edges" which contains a List of Tuples that represent edges between nodes. The Tuples are of ints that represent the index of the node in the "nodes" list.

Using this system, and adding a "coordinates" key with abstract drawing coordinates of each node for NetworkX, we can represent the counties of Connecticut like so:

def draw_map(planar_map, size, color_assignments=None):

        for i, e in enumerate(a_list):
            dct[i] = e
        return dct
    
    G = nx.Graph()
    
    labels = as_dictionary(planar_map[ "nodes"])
    pos = as_dictionary(planar_map["coordinates"])
    
    # create a List of Nodes as indices to match the "edges" entry.

    G.add_edges_from( planar_map[ "edges"])

assignFail: takes an input of a list indicating color assignment and return True if any node has no more valid assignment

def connectsTo(mp, n):
    return [x for (x,y) in mp['edges'] if y==n] + [y for (x,y) in mp['edges'] if x==n]

def unassignedNodes(assignment):
    return [not any([y==1 for y in x]) for x in assignment]

def assignComplete(assign):
    return all([any([y==1 for y in x]) for x in assign])

def assignFail(assign):
    return any([all([y==-1 for y in x]) for x in assign])

maxDegreeNode: The function takes the input of a list of integers representing the number of constraints each node has, and an assignment vector. Returns the index for the node with the highest number of constraint connections.

checkForward: The function performs the forward-checking aspect of the backtrack algorithm. It takes an input of an assignment vector and marks a color as specified by val in the assignment for the nodes specified in dest with -1, which eliminates those colors from consideration by the algorithm. The function returns None if the input assignment results in a inconsistent assignment, and the actual assignment if it doesnt.

constrainedBy: The function determines whether a choice of a color would be constrained by the current assignments. Takes an input of the assignment vector, a list of node indices, and a color indicated by val. The function loops through all the nodes and find if the same color is present in other already assigned node.

leastConstraining: The function list all the nodes that the input node is connected to by the number of "conflicts" that setting the indicated node to the indicated color would result in. The function returns a list of nodes that the algorithm could use to try different color for the backtrack algorithm.

def maxDegreeNode(degrees, assign):
    un = unassignedNodes(assign)
    tmp = [degrees[n] if x else -sys.maxsize for n,x in enumerate(un)]
    return tmp.index( max(tmp) )

def checkForward(inAssign, dests, val):
    assign = copy.deepcopy(inAssign)
    for d in dests:
        if not any([x==1 for x in assign[d]]):
            assign[d][val] = -1
    if assignFail(assign):
        return None
    else:
        return assign

def constrainedBy(assign, dests, val):
    for d in dests:
        try:
            color = assign[d].index(1)
            if color == val:
                return True
        except ValueError:
            pass
    return False

def leastConstraining(assign, dests, node):
    unassigned = unassignedNodes(assign)
    dests = [d for d in dests if unassigned[d]]
    
    vals = [n for n,x in enumerate(assign[node]) if x==0]
    conflicts = [0]*len(vals)
    for n,v in enumerate(vals):
        conflicts[n] = sum( [assign[d][v]==0 for d in dests] )
    
    # print(vals,conflicts) # print vals and no. of conflicts
    return [v for c,v in sorted(zip(conflicts,vals))]

Main Backtrack and Forward Checking Algorithm¶

The following functions implement the backtracking algorithm from fig. 6.5 on p. 215 of the textbook.

The color_map function first sets up the CSP by storing three extra set of information in the plannar map from the input. It stores a list of lists that indicates all the nodes that each individual node connects to, such that mp['dests'][i] stores the index of the node that the i-th node is connected to. The function also stores the number of constraints that each node has, such that mp['degrees'][i] is the number of constraints the i-th node shares with other node. It also stores the vector of strings representing colors, but this is only used for debugging trace.

The backtrack function is a recursive function that performs a DFS on the CSP tree and searches for a solution. It first selects a variable by picking the node with the least number of constraint connections that is yet without a color assignment. For the selected node, it tries each color selection based on the least constraining criteria, and makes an inference using forward checking. If the forward checking results in a consistent assignment, it recurs into a deeper level and attempts to find an assignment for the sub-tree. If a solution cannot be found, it returns None.

def color_map( mp, colors, trace=False):
    # get and store destinations of each node and degrees of each node
    mp['dests'] = [connectsTo(mp,n) for n,x in enumerate(mp['nodes'])] 
    mp['degrees'] = [len(x) for x in mp['dests']] #
    mp['colors'] = colors
    if trace:
        print('CSP starting point looks like: ')
        print(str(mp) + '\n')

    assign = [[0]*len(colors) for c in mp['nodes']] # color assignments
    out =  backtrack(assign, mp, trace)

    if out is not None:
        tmp = [(mp['nodes'][n],colors[x.index(1)]) for n,x in enumerate(out)]
        if trace:
            print('\nSolution color mapping is: ' + str(tmp))
        return tmp
    else:
        if trace:
            print('Cannot find solution.')
        return out

def backtrack(assign, mp, trace):
    if trace:
        print('Starting new recursion...')
    if assignComplete(assign):
        if trace:
            print('Assignment complete. Returning...')
        return assign
    nd = maxDegreeNode(mp['degrees'], assign) # select node of fewest degrees
    # nd = minRemValueNode(assign) # selected node by fewest rem. values
    if trace:
        print('Selected node ' + str(nd) + ' which has largest degree.')
    dests = mp['dests'][nd] # other nodes the selected node connects to
    cols = leastConstraining(assign, dests, nd) # colors sort by least conflict
    
    if trace:
        print('Node ' + str(nd) + ' connects to: ' + str(dests))
        print('Enumerate: ' + str(cols))

    for c in cols: 
        tmp = copy.deepcopy(assign) # temp. copy of color assignments
        if trace:
            print('Trying color #' + str(c) + ' (' + str(mp['colors'][c]) \
                 + ') for node ' + str(nd))

        if not constrainedBy(tmp, dests, c):
            if trace:
                print('Not constrained, adding to assignment')
            tmp[nd]= [1 if n==c else -1 for n,x in enumerate(tmp[nd])]
            # tmp[nd][c] = 1 # add current value to assignment
            infer = checkForward(tmp, dests, c)
            if infer is not None:
                if trace:
                    print('Forward checking is fine. New assignment:')
                    print(str(infer) + '\n')
                res = backtrack(infer, mp, trace)
                if res is not None:
                    return res
            else:
                if trace:
                    print('Forward checking failed, backtracking...\n')
        else:
            if trace:
                print('Contrained, trying another color.\n')

        assign[nd][c] = -1 # does not fit within constraint, set to off
    return None
